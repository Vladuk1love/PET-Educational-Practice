## 2. Установка

Перейдите в папку `./Practice` командой<br>

`cd .\Practice\`

Запустите `docker` образ командой <br>

`docker-compose up -d`

## 2.1 Ручная установка

### Backend:
    cd .\Practice\backend\  
    
    yarn

Запуск: `yarn run start:dev`  
### Frontend: 
    cd .\Practice\frontend\

    yarn 
    
Запуск: `yarn start`
